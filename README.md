# Caseflow Website

This folder contains the production site files prepared for a clean GitHub upload.

## Structure

- `index.html` - homepage
- `pages/` - secondary static pages
- `assets/css/` - site styles
- `assets/js/` - site interactions
- `assets/fonts/` - required web fonts
- `assets/img/` - brand and responsive image assets used by the site

## Notes

- Internal links use relative paths so the site works as a static upload.
- The package excludes mockups, raw source assets, and non-site reference files.
- Open `index.html` from this folder or deploy the whole folder root as a static site.
